# Claude Code Assistant — Alexa Skill

## Overview

Claude Code Assistant is an Alexa skill powered by Anthropic's Claude AI.
It helps developers build new features, fix bugs, improve code quality,
and provides step-by-step implementation guides — all with complete
before/after code comparisons and exact line numbers.

---

## Key Features

- Build complete production-ready features from voice commands
- Fix bugs with root cause analysis and targeted code fixes
- Improve code quality (performance, accessibility, security, responsiveness)
- Step-by-step implementation guides with file names and line numbers
- Before/after code comparison for every change
- Permission system — Claude always asks before making changes
- Instruction diversion — redirect Claude mid-task to a different approach
- Multi-turn conversation with full session memory
